Input Text- criar campos de textos básicos- maxlength: define uma valor máximo, minlength: define valor mínimo, size: define quantos caracteres o campo de entrada deve ter

Input password- cria um campo de senha onde o texto fica obscuro- maxlength: define uma valor máximo, minlength: define valor mínimo

Input checkbox - caixas que são marcadas quando escolhe uma opção- checked: define uma caixa para ser marcada como padrão

Choose your monster's features:

Input botão- são botões simples- value: define o texto que aparecerá no seu botão

Input Date- define uma caixa de texto onde é possivel o usuário inserir uma data- min: data mínimo, max: data máximo, value: valor padrão

Input Radio- caixas que são marcadas quando escolhe uma opção podendo ser marcada somente 1 opção
